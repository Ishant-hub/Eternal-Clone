# Eternal-Clone

Eternal Clone is a fully responsive front-end project built to practice and demonstrate HTML, CSS, and basic JavaScript skills.
It recreates the modern and elegant design of the official Eternal website, showcasing strong attention to layout structure, design aesthetics, and web animations.

⚠️ Disclaimer:
This project is strictly for educational and portfolio purposes. All content, brand names, and images belong to their respective owners.

✨ Features

🔹 Responsive navigation bar

🔹 Smooth section-based layout (Home, Vision, Careers, Contact, etc.)

🔹 Interactive loading animation

🔹 Hover glow effects on headings and buttons

🔹 Modern and minimal UI inspired by Eternal’s branding

🔹 Built using only HTML, CSS, and JavaScript

🛠️ Technologies Used
Technology	Purpose
HTML5	Structure and semantic markup
CSS3	Styling, layout, and animations
JavaScript	Dynamic interactions (e.g., loader, hover effects)
GitHub 	Deployment and hosting
